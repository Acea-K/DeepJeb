> This .skill file is a structured world knowledge base for LLM retrieval.

## 1. GAME OVERVIEW

**Version scope:** This document is primarily based on KSP 1.12.5 (the final KSP1 release). KSP2 is a separate game with different rules and is not covered here.

| Property | Value |
|----------|-------|
| Developer | Squad (Mexico) |
| Publisher | Private Division (Take-Two Interactive) |
| Initial release | 2011 (early access), 2015 (v1.0 full release) |
| Latest version | 1.12.5 ("The Once And Future Kerbal Kind") |
| Engine | Unity (upgraded to Unity 2019.2 in v1.8) |
| Platforms | Windows, macOS, Linux, PlayStation 4/5, Xbox One/Series X\|S |
| DLC | Making History (v1.4+, March 2018), Breaking Ground (v1.7.1+, May 2019) |

### Game Modes
- **Sandbox:** All parts unlocked, unlimited funds. Pure creative play.
- **Science:** Must collect science points to unlock parts. No financial constraints.
- **Career:** Complete contracts for funds and reputation, collect science for parts. Full program management.
