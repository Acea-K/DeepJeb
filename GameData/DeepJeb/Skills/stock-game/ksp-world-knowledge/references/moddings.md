# Modding

## Essential Mods
- **Kerbal Engineer Redux (KER):** Δv, TWR, orbital data readouts
- **Module Manager:** Dependency for almost every other mod (see [module-manager.skill] for syntax guide)
- **Kopernicus:** Custom planetary systems (version-locked!)
- **Scatterer:** Atmospheric scattering, sunlight effects
- **Environmental Visual Enhancements (EVE):** Clouds, city lights
- **Real Solar System (RSS) / Realism Overhaul (RO):** Real-world scale (hardcore)
- **KSP-AVC:** Mod version checker
- **Kerbal Attachment System (KAS) / Kerbal Inventory System (KIS):** EVA tools
- **CKAN:** Mod manager (may be slow in mainland China)
- **Textures Unlimited:** Fixes DX11 texture issues

## Mod Installation
1. Download mod (SpaceDock, GitHub, CurseForge, or Forum)
2. Extract to `<KSP Folder>/GameData/`
3. Required dependencies: Module Manager should auto-install
4. Restart game
5. **Do NOT** use "all-in-one packs" — outdated and potentially pirated

## Common Mod Issues
- **Version incompatibility:** Most mods break after major KSP updates
- **Memory use:** Visual mods consume significant RAM
- **Kopernicus is version-locked:** Must match exact KSP version
- **Principia (n-body physics):** Locked to specific versions (1.5.1–1.7.3)
- **Chinese translation + Kopernicus:** May conflict

## Why "All-in-One Packs" Are Discouraged
- Often contain pirated game files (illegal)
- May violate mod license agreements
- "Release and abandon" — no updates or support
- Can ruin the experience for new players
