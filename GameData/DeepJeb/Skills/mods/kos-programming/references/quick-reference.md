# kOS Quick Reference

## Essential Commands

| Command | Description |
|---------|-------------|
| `PRINT "text".` | Output text to terminal |
| `SET var TO value.` | Assign variable |
| `LOCK var TO expression.` | Lock variable (updates each frame) |
| `UNLOCK var.` | Unlock variable |
| `CLEARSCREEN.` | Clear terminal |
| `RUN script.ks.` | Run script file |
| `RUNPATH("script.ks", args).` | Run with arguments |
| `LIST.` | List available commands |
| `LIST BODIES IN b.` | List celestial bodies |
| `LIST VESSELS IN v.` | List vessels |
| `LIST PARTS IN p.` | List parts |
